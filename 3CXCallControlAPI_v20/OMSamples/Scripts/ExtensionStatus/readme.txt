When called from the extension, the status of the extension will be switched to the requested one
NOTE: temporary status override will not be reset
#0 - set Available status
#1 - set Away status
#2 - set Out of office (a.k.a DND) status 
#3 - set Custom 1 status (custom available status)
#4 - set Custom 2 status (custom out of office status)
