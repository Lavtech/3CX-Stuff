      BlackListEntry Properties

      Properties

      AddedBy Who add it and who should manage it. "0"(or empty)- administrator of PBX using ManagementConsole, "1" - whitelist for MCU servers, "2" - whitelist for Webmeeting servers, "3" - blacklist downloaded from 3CX site, "4" - added by MC Whitelist feature
      BlockType type of block. if set to 0 - This IP is blocked. (default value)
      if set to 1 - This IP always allowed. (should not be automatically blocked) by any component
      Description description of the entry.
      ExpiresAt the entry is valid until this time
      HasAttached Indicates that the object has attached data for saving transaction
      (Inherited from IOMSnapshot)
      ID each object has unique ID. It is the same as the value returned by GetHashCode. (legacy) new object has ID=0. After successful call to save snapshot gets persistent ID.
      (Inherited from IOMSnapshot)
      IPAddr blacklisted IP. In combination with IPMask may specify subnets.
      IPMask if the entry specifies single IP, mask should be specified as "255.255.255.255". In combination with IPAddr may specify subnets
