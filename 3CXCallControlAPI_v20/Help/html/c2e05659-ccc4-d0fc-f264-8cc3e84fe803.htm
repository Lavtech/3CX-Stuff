      Remarks

      Race condition with update:
      if statistics table is initialized using InitializeStatistics(String), the cache of the table will be not be updated and CreateStatistics(String, String) will continue to return previous state of the object until update notification will not come from the server and will not be processed by backend. method update(Boolean) is added to give ability to guarantee that call to CreateStatistics(String, String) will return recent update after update will finish.
